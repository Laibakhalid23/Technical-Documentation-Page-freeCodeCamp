Introduction

Python is a high-level, interpreted programming language with a clear and readable syntax. It was created by Guido van Rossum and first released in 1991.

Python supports multiple programming paradigms including procedural, object-oriented, and functional programming. It is known for its use in a wide variety of domains such as web development, data science, artificial intelligence, and automation.

Getting Started

To get started with Python, download and install it from the official Python website. Python 3.x is the latest version and is recommended for all new projects.

After installation, open a terminal and verify your installation:

python --version

You can start writing Python programs in any text editor and save the file with a `.py` extension. To run your script, use:

python your_script_name.py

Control Flow

Python provides control flow statements such as `if`, `elif`, and `else` to allow decision-making in your programs.

Example:

x = 5
if x > 0:
  print("Positive")
else:
  print("Negative")

Indentation is very important in Python and defines blocks of code.

Functions

Functions are reusable blocks of code that perform a specific task. In Python, you define a function using the def keyword.

Example:

def greet(name):
  return "Hello, " + name

print(greet("Alice"))

Functions can take parameters, return values, and help organize your code better for reusability and clarity.
